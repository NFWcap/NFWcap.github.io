Git关联远程仓库

对github或者其他git仓库中文件和文件名进行修改、删除操作

和远程仓库建立关联

在git bash中先建立一个用于和远程仓库建立连接的目录：

$mkdir learnpy

$ cd learnpy

找到远程仓库的URL，这里是

http://git.sodevel.com/wll_admin/python.git

进行关联

$ git remote add origin http://git.sodevel.com/wll_admin/python.git

这个时候本地的learnpy已经关联到了远程仓库
