Hexo博客使用指南II：向博客中插入图片

如何向文章中插入图片

插入图片

  • 首先我们需要用到的插件CodeFalling/hexo-asset-image
  • 确认在_config.yml中post_assert_folder选项为：true
  • 在hexo目录下执行下面的命令，安装插件
    • npm install https://github.com/CodeFalling/hexo-asset-image --save
  • 这样在执行hexo new命令时，会在_post目录下生成一个与文件名相同的文件夹，可以将插入的图片文件放入此文件夹下

  • 在我使用的makedown语法编辑器Typora中，可以先将图片放入同名文件夹后，在拖入编辑器中，可自动生成图片

  • Tips

    • 在xxxx.md中想引入图片时，先把图片复制到xxxx这个文件夹中，然后只需要在xxxx.md中按照markdown的格式引入图片：

      注意： xxxx是这个md文件的名字，也是同名文件夹的名字。只需要有文件夹名字即可，不需要有什么绝对路径。你想引入的图片就只需要放入xxxx这个文件夹内就好了。
